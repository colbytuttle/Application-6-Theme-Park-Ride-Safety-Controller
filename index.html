Application 6 Project

Project Overview

This project simulates a Theme Park Ride Safety Controller using an ESP32 and FreeRTOS to demonstrate a real-time system. The additional hardware includes LEDs, which represent different system statuses, such as active ride, warning indicator for safety concerns regarding speed, and an emergency signal. Uses a potentiometer to simulate ride conditions, such as speed and load, as well as a push button which represents and emergency stop for the ride, regardless of any other system components. Real-time tasks are used to ensure deadlines are met for both the hard and soft real-time tasks within the system.
